# Demo Video Quick Reference

**One-page guide for creating your demo video**

---

## ⏱️ Timeline (3 minutes total)

| Time | Section | Content |
|------|---------|---------|
| 0:00-0:20 | Intro | Project overview, AWS services |
| 0:20-0:50 | Upload | Upload document, enter symptoms |
| 0:50-1:20 | Process | Explain AI pipeline |
| 1:20-2:20 | Results | Show recommendations |
| 2:20-2:50 | Tech | AWS services, innovation |
| 2:50-3:00 | Close | GitHub link, demo link |

---

## 📝 Script Highlights

**Introduction**:
> "This is the Document Policy Processor - an AI-powered solution that helps users understand which insurance policies apply to their medical situation."

**Demo**:
> "I'm uploading a medical report and describing my symptoms: 'Diagnosed with Type 2 diabetes, requiring insulin therapy and regular monitoring.'"

**Results**:
> "The system found three relevant policies. The top match is 'Comprehensive Health Plus' with 94% confidence - it covers diabetes management with no exclusions."

**Conclusion**:
> "The Document Policy Processor makes insurance accessible, accurate, and instant. Check out the code on GitHub!"

## 🎬 Recording Checklist

**Before Recording**:
- [ ] Close unnecessary apps
- [ ] Disable notifications
- [ ] Test application works
- [ ] Have sample document ready
- [ ] Practice script 2-3 times

**Recording**:
- [ ] Speak clearly, moderate pace
- [ ] Move mouse slowly
- [ ] Pause on important info
- [ ] Record 2-3 takes

**After Recording**:
- [ ] Review all footage
- [ ] Check audio quality
- [ ] Verify under 3 minutes

---

## ✂️ Editing Checklist

- [ ] Trim dead space
- [ ] Remove mistakes
- [ ] Add title slide (0:00-0:05)
- [ ] Add closing slide (2:50-3:00)
- [ ] Add text overlays for key points
- [ ] Normalize audio levels
- [ ] Export: 1080p MP4, H.264
