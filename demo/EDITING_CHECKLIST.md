## Editing Workflow

### Step 1: Import Footage

- [ ] Create new project in your editing software
- [ ] Import all recorded clips
- [ ] Import any additional assets (title slides, diagrams, logos)
- [ ] Organize clips in folders or bins by section
- [ ] Review all clips to identify best takes

### Step 2: Create Rough Cut

- [ ] Drag clips to timeline in order
- [ ] Trim beginning and end of each clip (remove dead space)
- [ ] Remove any mistakes or long pauses
- [ ] Ensure smooth flow between sections
- [ ] Check total duration (should be under 3 minutes)

### Step 3: Fine-Tune Timing

- [ ] Watch entire video from start to finish
- [ ] Adjust timing of each section
- [ ] Ensure adequate time to read on-screen text
- [ ] Remove any awkward pauses
- [ ] Speed up slow sections if needed (1.1x-1.2x speed)
- [ ] Verify final duration is under 180 seconds

### Step 4: Add Enhancements

- [ ] Add title slide at beginning
- [ ] Add closing slide at end
- [ ] Add transitions between sections (subtle, not distracting)
- [ ] Add text overlays for key points
- [ ] Add arrows or highlights to draw attention
- [ ] Adjust audio levels for consistency

### Step 5: Polish

- [ ] Color correction (if needed)
- [ ] Audio enhancement (noise reduction, normalization)
- [ ] Add background music (optional, keep it subtle)
- [ ] Add sound effects (optional, use sparingly)
- [ ] Final timing adjustments

### Step 6: Export

- [ ] Export at 1080p resolution
- [ ] Use H.264 codec for compatibility
- [ ] Export as MP4 format
- [ ] Review exported video before uploading

## Essential Edits

### 1. Trim Dead Space

**What**: Remove silence or inactivity at beginning and end of clips

**How**:
- Cut first 1-2 seconds of each clip (before you start speaking)
- Cut last 1-2 seconds of each clip (after you finish)
- Remove long pauses between sentences (keep natural rhythm)

**Why**: Keeps video tight and professional

### 2. Remove Mistakes

**What**: Cut out any errors, stumbles, or retakes

**How**:
- Identify mistakes while reviewing footage
- Use razor/cut tool to split clip at mistake
- Delete the mistake section
- Smooth the transition with a quick fade

**Why**: Maintains professional appearance

### 3. Adjust Pacing

**What**: Ensure video moves at appropriate speed

**How**:
- Speed up slow sections by 10-20% (right-click → speed/duration)
- Slow down fast sections if needed
- Add brief pauses (0.5-1 second) between major sections
- Ensure viewers have time to read on-screen text

**Why**: Keeps viewers engaged, ensures comprehension

### 4. Sync Audio and Video

**What**: Ensure voiceover matches on-screen actions

**How**:
- Align audio with corresponding video
- Use markers or visual cues to sync
- Adjust timing of audio or video clips as needed
- Test by watching multiple times

**Why**: Professional videos have perfect sync

### 5. Normalize Audio Levels

**What**: Ensure consistent volume throughout

**How**:
- Select all audio clips
- Apply audio normalization (target: -3dB to -6dB)
- Manually adjust clips that are too loud or quiet
- Use audio compressor to even out levels

**Why**: Prevents jarring volume changes

## Polish and Enhancement

### Title Slide (0:00-0:05)

**Content**:
- Project name: "Document Policy Processor"
- Subtitle: "AI-Powered Insurance Policy Analysis"
- Your name or team name
- "Built for AWS AI for Bharat Hackathon"

**Design Tips**:
- Use clean, professional font (Arial, Helvetica, Roboto)
- Include AWS logo (if permitted by guidelines)
- Use brand colors or professional color scheme
- Keep it simple - not too much text

**Duration**: 3-5 seconds

### Text Overlays

Add text overlays to emphasize key points:

**Examples**:
- "AWS Textract → OCR"
- "Semantic Embeddings"
- "94% Confidence"
- "No Exclusions Found"

**Best Practices**:
- Use large, readable font (minimum 48pt)
- Place in lower third of screen (doesn't cover important content)
- Use contrasting colors (white text on dark background or vice versa)
- Animate in/out smoothly (fade or slide)
- Display for 2-3 seconds
- Don't overuse - only for key points

### Transitions

**Between Sections**:
- Use simple transitions (fade, dissolve)
- Duration: 0.5-1 second
- Avoid flashy transitions (wipes, spins) - they look unprofessional

**Best Practices**:
- Use same transition throughout for consistency
- Don't use transitions between every clip
- Only use between major sections

### Arrows and Highlights

**When to Use**:
- Drawing attention to specific UI elements
- Highlighting important text or numbers
- Showing where to click

**How to Add**:
- Use annotation tools in editing software
- Or create arrows in PowerPoint/Keynote and export as PNG
- Animate in when needed, animate out when done
- Use bright colors (red, yellow) for visibility

**Best Practices**:
- Don't overuse - only when necessary
- Remove when no longer needed
- Keep style consistent

### Background Music (Optional)

**When to Use**:
- Can add energy and professionalism
- Keep volume low (20-30% of voiceover)
- Use only if it enhances, not distracts

**Where to Find**:
- YouTube Audio Library (free, no attribution required)
- Epidemic Sound (paid subscription)
- Artlist (paid subscription)
- Free Music Archive (free, check licenses)

**Best Practices**:
- Choose upbeat, professional instrumental music
- Avoid music with lyrics (competes with voiceover)
- Fade in at beginning, fade out at end
- Keep volume low - voiceover should be primary
- Ensure you have rights to use the music

### Closing Slide (2:50-3:00)

**Content**:
- "Document Policy Processor"
- "GitHub: github.com/yourusername/document-policy-processor"
- "Live Demo: your-demo-url.com"
- "Thank you!"

**Design Tips**:
- Match style of opening slide
- Make URLs clearly readable
- Consider adding QR codes for easy access
- Include your contact info (optional)

**Duration**: 5-10 seconds

## Export Settings

### Recommended Settings for YouTube

**Format**: MP4 (H.264)
**Resolution**: 1920x1080 (1080p)
**Frame Rate**: 30 fps
**Bitrate**: 8-12 Mbps (higher is better quality)
**Audio**: AAC, 192 kbps, 48 kHz

### Export Settings by Software

**DaVinci Resolve**:
1. File → Deliver
2. Format: MP4
3. Codec: H.264
4. Resolution: 1920x1080
5. Frame Rate: 30
6. Quality: Automatic or 8000 kb/s
7. Click "Add to Render Queue"
8. Click "Start Render"

**iMovie**:
1. File → Share → File
2. Resolution: 1080p
3. Quality: High
4. Compress: Better Quality
5. Click "Next" and save

**Windows Video Editor**:
1. Click "Finish video"
2. Select "High 1080p"
3. Click "Export"

**Kapwing**:
1. Click "Export video"
2. Select 1080p resolution
3. Click "Export"
4. Download when ready

### File Size

**Target**: 100-500 MB for 3-minute video
- If too large (>500 MB): Reduce bitrate
- If too small (<50 MB): Increase bitrate for better quality

## Common Editing Mistakes to Avoid

### ❌ Don't Do This

1. **Too many transitions**: Stick to simple fades
2. **Flashy effects**: Avoid spinning text, excessive animations
3. **Loud background music**: Keep it subtle or skip it
4. **Too much text on screen**: Keep it minimal and readable
5. **Inconsistent style**: Use same fonts, colors, transitions throughout
6. **Going over 3 minutes**: Judges may stop watching
7. **Poor audio quality**: Fix audio issues before finalizing
8. **Unreadable text**: Ensure all text is large and clear
9. **Rushed pacing**: Give viewers time to absorb information
10. **No call to action**: Always include GitHub and demo links

### ✅ Do This Instead

1. **Simple, professional transitions**: Fade or dissolve
2. **Minimal effects**: Only when they add value
3. **Balanced audio**: Voiceover is primary, music is background
4. **Key points only**: Highlight only the most important information
5. **Consistent branding**: Same style from start to finish
6. **Under 3 minutes**: Aim for 2:45-2:55 to be safe
7. **Clear audio**: Normalize, reduce noise, ensure clarity
8. **Large, readable text**: Minimum 48pt font
9. **Comfortable pacing**: Not too fast, not too slow
10. **Clear next steps**: Tell viewers where to find more info

## Time-Saving Tips

1. **Use templates**: Create title slide template and reuse
2. **Keyboard shortcuts**: Learn shortcuts for your editing software
3. **Batch operations**: Apply effects to multiple clips at once
4. **Presets**: Save export settings as preset for future use

6. **Plan before editing**: Know what you want before you start
7. **Use markers**: Mark important points while reviewing footage
8. **Duplicate project**: Save versions as you work (v1, v2, v3)
